# Анализ поисковых запросов Яндекс.Картинок  
В этом проекте проведён анализ поисковых запросов к сервису Яндекс.Картинки. Основная цель — проверить гипотезу о том, что интересы пользователей мобильных устройств (touch) и компьютеров (desktop) существенно различаются  

Задачи исследования:   
- Определить временной диапазон данных  
- Проанализировать популярность запросов по платформам   
- Выявить различия в тематиках и поведении пользователей   
- Сформулировать выводы и рекомендации для команды продукта  

## Инструменты и технологии  
- Python - pandas, pandasql, scikit-learn, matplotlib / seaborn  
- Jupyter Notebook  
- Yandex Datalens  

## Визуализации  
В проекте представлены следующие графики:  
- Динамика запросов по часам (платформы)  
- Топ-10 запросов по платформам

Распределение тематик (кластеризация KMeans)  
  - Сравнение долей категорий (heatmap / stacked bar)

## Ключевые выводы  
Проведённый анализ подтверждает гипотезу о том, что интересы пользователей поиска по картинкам существенно отличаются в зависимости от типа устройства.  
|   Тип устройства  | Тип запросов | Время активности |
|---------------|-----------------------------|---------|
| Desktop | Работа, учеба | 8:00 – 17:00 (рабочее время)  |
| Touch | Поздравления, досуг, погода | 05:00 – 08:00 и 17:00 – 19:00 (утро/вечер)  |
